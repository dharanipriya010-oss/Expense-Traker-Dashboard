 Expense Tracker Dashboard

A full-stack web application designed to help users seamlessly log, track, and analyze their daily finances.

 🚀 Features
   Interactive Dashboard: Real-time visual data summaries of income vs. expenses.
   Transaction Management: Easily add, update, and delete expense logs.
   Smart Categorization: Group spending into custom categories (e.g., Food, Rent, Entertainment).
   Analytics: Clean graphs and charts to track monthly budgeting trends.

 🛠️ Tech Stack
   Frontend: React.js, HTML5, CSS3, JavaScript
   Backend: Node.js, Express.js
   Database: MongoDB
   API Testing
